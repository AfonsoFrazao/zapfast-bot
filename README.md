
# ZapFastBot

Bot de Telegram para venda de números virtuais via integração com a API da 5sim.

## Como usar

1. Edite o arquivo `main.py` e substitua:
   - `COLE_SEU_TOKEN_AQUI` pelo seu token do BotFather
   - `COLE_SUA_API_KEY_AQUI` pela sua API da 5sim.net

2. Suba este código no [https://render.com](https://render.com) usando a opção de "Public Git Repository"

3. Use o seguinte comando para iniciar:
   ```
   python3 main.py
   ```

Pronto! Seu bot estará ativo e pronto para começar a vender.
